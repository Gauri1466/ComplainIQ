# ComplainIQ — Setup Guide
# ========================
# iDEA 2.0 Hackathon · PS5 · Team Avinya

## INSTALL (run once)
pip install streamlit plotly anthropic

## RUN
streamlit run complainiq_app.py

## THEN
1. Open browser at http://localhost:8501
2. Go to Settings → enter your Claude API key
3. Go to Inbox → click Open on any complaint
4. Click "Analyze with AI" → watch the magic!

## GET YOUR API KEY
https://console.anthropic.com → Create API Key (free tier available)

## DEPLOY FREE (share with judges)
1. Push to GitHub
2. Go to share.streamlit.io
3. Connect repo → deploy in 2 minutes
4. Share the URL with judges!
